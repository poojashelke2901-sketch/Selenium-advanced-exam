# Selenium Advanced Assignment

## Project Description
This project is created as part of Selenium Advanced Test Assignment using Java, Selenium 4, and TestNG.

---

##  Tech Stack
- Java
- Selenium 4
- TestNG
- Maven
- Chrome Browser

---

## Project Structuresrc/test/java/tests → Test Classes
src/test/resources → testng.xml
screenshots → Execution screenshots
pom.xml → Dependencies---

##  How to Run the Project

1. Open project in Eclipse  
2. Right-click on `testng.xml`  
3. Click → Run As → TestNG Suite  

---

## Test Scenario Covered

1. Navigate to https://www.testmuial.com  
2. Apply explicit wait  
3. Scroll to "Explore Agentic Clouds"  
4. Click and verify new tab  
5. Handle window handles  
6. Verify URL  
7. Scroll to "Seamlessly Scale..."  
8. Click "Try Now For Free"  
9. Verify page title  
10. Close window  
11. Print window count  
12. Navigate to blog  
13. Click Community and verify URL  
14. Close browser
